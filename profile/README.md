## Governance

- 具体业务代码、部署和数据资产留在对应 BU。
- 跨 BU 的战略、组合、制度和决策进入 `President-Office`。
- 平台底座和代码生成资产进入 `Business-Unit-for-Platform`。
- 生产部署默认按多机模式设计，数据库/缓存不以 `127.0.0.1` 作为生产默认。
